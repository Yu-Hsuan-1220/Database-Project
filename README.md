# NYCU Introduction to Database Systems

A Leaguage of Legends assistant **web server** based on **database** with the following functions: 

## Funcion:
1. User sign-up and login system.

2. Retreive information and **Tier lists** for each hero from S12.1~S13.

3. Add and search your personal **win rates** of different **lanes** and **heroes**.

4. Submit personal evaluations of different heroes and search for evaluations from other users.

5. Search win rates based on different scenarios.
